## Experiment 1: Allocentric vs. Egocentric Hypothesis

### Conditions

| Condition | Description | Delay |
|---|---|---|
| Baseline-stationary | Observer stands at home base, judges target | 12 or 60 sec |
| Walking (12 sec delay) | Walk 1.5 m, wait 12 sec, judge target | 12 sec |
| Walking (60 sec delay) | Walk 1.5 m, wait 60 sec, judge target | 60 sec |

### Key Results

| Comparison | Finding | Direction | Significance |
|---|---|---|---|
| Walking (12 sec) vs Baseline-stationary | Judged locations significantly nearer in walking | Supports allocentric | Significant |
| Walking (60 sec) vs Baseline-stationary | Much smaller separation than 12 sec | Allocentric effect decays | Reduced |
| Intrinsic bias shift | ~1.35 m horizontal shift | Consistent with 1.5 m walk | Estimated from Fig. 3c |

- Fig. 3c: Average judged target locations (n=8) plotted for all conditions
- Filled green triangles (12 sec walking) are significantly nearer than red circles (baseline-stationary)
- Open green triangles (60 sec walking) show smaller separation from baseline
- Blue and gray intrinsic bias curves fitted by eye show ~1.35 m horizontal shift
- Black circle on vertical axis represents average observer eye height
- Fig. 3e-f: Angular declination plots showing judged vs physical angles

## Experiment 2: Divided Attention During Walking

### Conditions

| Condition | Description |
|---|---|
| Baseline-stationary | Standard stationary judgment |
| Divided-attention-walking | Walk 1.5 m while performing cognitive task, then judge target |

### Key Results

| Comparison | Finding | Significance |
|---|---|---|
| Divided-attention-walking vs Baseline-stationary | Judged locations similar (overlapping) | Not significant |
| Implication | Attention distraction disrupts path-integration | Intrinsic bias travels with observer |

- Fig. 3d: Average judged locations (n=8) from divided-attention-walking and baseline-stationary conditions are similar
- Both conditions fitted by the same intrinsic bias curve
- Divided attention prevents the path-integration mechanism from anchoring the intrinsic bias at the home base

## Experiment 5: Stepladder Descent (Anisotropic Path-Integration)

### Design Parameters

| Parameter | Value |
|---|---|
| Stepladder height | 0.6 m |
| Participants | n=9 |
| Manipulation | Observer descends from stepladder in the dark |

### Conditions

| Condition | Description |
|---|---|
| Stepladder | Observer descends 0.6 m vertically, judges target from floor |
| Ground-level baseline | Observer stands on floor, judges target |

### Predictions

| Hypothesis | Horizontal Coordinate | Vertical Coordinate |
|---|---|---|
| Isotropic path-integration | Stays at home base | Stays at stepladder height |
| Horizontal-only path-integration | Stays at home base (floor location) | Shifts down with observer |

### Key Results

| Comparison | Horizontal | Vertical | Interpretation |
|---|---|---|---|
| Stepladder vs Ground-level | No horizontal shift | Vertical shift observed | Horizontal-only path-integration |
| Implication | Odometer only gauges horizontal distance | Vertical motion not integrated | Anisotropic |

- Fig. 5a: Prediction diagram for horizontal-only path-integration
- Fig. 5c: Average results (n=9) showing anisotropic pattern
- The path-integration mechanism operates exclusively along the horizontal direction
- This finding parallels anisotropic spatial memory observed in desert ants

## Intrinsic Bias Properties

| Property | Finding |
|---|---|
| Shape | Curved surface representation |
| Default behavior | Used to localize objects in the dark |
| Integration | Combines with visible depth cues (e.g., texture gradients) |
| Coordinate system | Allocentric (world-centered) |
| Anchoring | Fixed at home base location on ground |
| Path-integration | Updates with observer motion (vestibular signals) |
| Attention dependency | Requires attentional resources during locomotion |
| Temporal decay | Effect diminishes from 12 sec to 60 sec post-walking |
| Directionality | Encodes direction of travel (forward vs backward) |
| Anisotropy | Horizontal motion only (no vertical integration) |

## Theoretical Framework

| Concept | Description |
|---|---|
| Ground-based spatial coding | Target locations judged with respect to ground surface |
| Intrinsic bias | Implicit curved surface representation in the dark |
| Allocentric hypothesis | Spatial reference fixed to world coordinates |
| Egocentric hypothesis | Spatial reference moves with observer |
| Path integration | Mechanism for tracking displacement during self-motion |
| Allocentric advantage | Reduces computational redundancy during locomotion |
